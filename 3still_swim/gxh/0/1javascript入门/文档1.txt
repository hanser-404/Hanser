学习要点 
  1：什么是javascript
   js 诞生于 1995年  他们的目的很简单 就是为了 实现一个功能 就是表单验证
   早些年 表单验证是通过服务器端 进行验证的 
经过多年的发展：js  从一个简单的输入验证 成为一门强大的编程语言。学习使用 js的时候 比c 或者java 简单一些  但是 如果想真正精通js  那需要的时间就漫长了  我看红皮书 看了 8个月  看各类文章 看了 2年  
javascript 是一种 具有面向对象能力的，解释性的程序设计语言，他就是基于对象和事件驱动具有一定安全性的客户端脚本语言 ，因为他不需要在一个语言环境下运行，而只需要有支持他的浏览器即可 他主要的目的是 验证发送服务器端数据 增加web前端的互动性 加强用户体验
  2：javascript 的特点
	松散性
      js语言核心 与 C C++ JAVA相似  比如 条件语句判断  循环  运算符
      但是他却是一种松散类型的语言  也就是说  他的变量 不具备一个明确的类型
	对象属性
       js中 对象把属性名映射为任意的属性值 ，他的这种方式很像 哈希表 或者  关联数组 ，他不像 c中的结构体或者c++ java 中的对象
        继承机制
      js中的面向对象 继承机制是基于原型的  这和另外一种self语言 很像
       和c++，java 的集成完全不同  
  3：js历史
	1992年 有一家公司  Nombas 开发一种叫做 C--的一种嵌入式脚本语言， 
后来 这个公司 觉得这个名字太晦气了  最后给改名叫做scriptEase 而这种可以嵌入网页中的脚本的理念 成为因特网的一个重要的基石
1995年 当时工作在 Netscape（网景）公司的 布兰登为了解决类似于 向服务器提交数据之前的验证 这个问题  联合sun公司 开发了一个成为 Livescript 这个脚本语言
